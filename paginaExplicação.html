🧠 O que é o DOM?

DOM significa Document Object Model. É a estrutura criada pelo navegador para representar o HTML.

O JavaScript utiliza o DOM para acessar e modificar os elementos da página.

<body> <h1>Título</h1> <button>Clique</button> </body>

O navegador transforma isso em uma árvore de elementos:

BODY ├── H1 └── BUTTON

🔗 Relação entre HTML, CSS e JavaScript

Tecnologia Função
HTML Estrutura da página
CSS Aparência e estilo
JavaScript Comportamento e interatividade
DOM Ponte entre HTML e JavaScript

🔌 Como o JavaScript acessa o DOM

O JavaScript usa o objeto document para acessar a página.

// Acessa o documento inteiro document

Exemplo buscando um elemento pelo ID:

// Busca elemento pelo ID document.getElementById("titulo");

🧪 Exemplo Completo

<h1 id="titulo">Olá</h1> <script> // Captura o elemento const titulo = document.getElementById("titulo"); // Altera o texto titulo.innerText = "Texto alterado pelo JavaScript"; </script>

⚙️ Eventos

Eventos são ações realizadas pelo usuário:

  • Cliques
  • Digitação
  • Carregamento da página
elemento.addEventListener("click", function(){ alert("Clicou!"); });

🛠️ Manipulação do DOM

O JavaScript consegue alterar texto, estilos e classes.

// Alterando texto elemento.innerText = "Novo texto"; // Alterando cor elemento.style.color = "red"; // Manipulando classes elemento.classList.remove("bloqueada"); elemento.classList.add("ativa");
